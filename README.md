ScrabbleDabble
=====================================

##About:

A programming challenge using basic Javascript for Epicodus Summer 2014.

##Author:

* [Christian Danielsen] (https://github.com/cdanielsen) and Kyle Boe.

##Description:

A program to calculate the scrabble score of a given word.

##Local Implementation:

Clone onto local machine and open index.html from the root directory in your favorite browser.
